# Slideshow Mode - Quick Reference

## Start Slideshow
Click **"▶ Slideshow"** button in the header

## Controls

### Show Controls
- **Move mouse** over the page
- Controls appear at bottom
- Page info appears at top-right
- Exit hint appears at bottom-left

### Buttons (appear on hover)
| Button | Action |
|--------|--------|
| ← Prev | Previous page |
| Next → | Next page |
| ✕ Exit | Exit slideshow |

### Speed
- **Fixed**: 3 seconds per page
- **Consistent**: No adjustment needed
- **Optimized**: For reading focus

## Display

### What You See
- Full-screen dark overlay
- Large PDF page display
- Page info (top right, on hover)
- Exit hint (bottom left, on hover)
- Controls (bottom center, on hover)

### What Happens
- Pages auto-advance every 3 seconds
- Progress automatically saves
- Slideshow ends at last page

## Tips

✅ **Good Practices**
- Use for focused reading
- Pause for complex content
- Use dark theme for comfort
- Take breaks between chapters

❌ **Avoid**
- Reading in bright light
- Skipping pages manually
- Long sessions without breaks

## Common Tasks

### Show Controls
1. Move mouse over the page
2. Controls appear at bottom
3. Click to navigate or exit

### Go Back a Page
1. Move mouse to show controls
2. Click "← Prev" button
3. Continue slideshow

### Exit Slideshow
- Click "✕ Exit" button, OR
- Press ESC key, OR
- Click anywhere on page

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Won't start | Ensure PDF loaded |
| Controls not showing | Move mouse over page |
| Pages not advancing | Check if PDF has pages |
| Performance slow | Close other tabs |
